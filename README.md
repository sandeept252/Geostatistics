# Geostatistics by @sandeept252

This repository consists text, Excel, jupyter notebook files and PDFs related to Geostatistics. The content is primarily based on the syllabus for M.Sc. Engineering Geology at Tribhuvan University.
It closely follows the book *Statistics and Data Analysis in Geology* by John C. Davis.

Datasets are available at https://bcs.wiley.com/he-bcs/Books?action=chapter&bcsId=1479&itemId=0471172758&chapterId=4060

The datasets are analyzed using the following softwares: 
	1. MS-Excel
	2. Python
	3. R
